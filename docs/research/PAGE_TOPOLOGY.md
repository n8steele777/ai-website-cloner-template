# Off Menu Homepage Page Topology

## Sections
1. Fixed header
2. Full-screen hero orbit section
3. Footer contact strip

## Overlay Layers
- Header sits above all content
- Hero orbit sits behind headline and control overlays
- Active title and navigation arrows float over the orbit
- Footer is flow content below the hero

## Interaction Model
- Header: click-driven
- Hero orbit: click-driven in this clone
- Footer: static with standard link hover behavior
