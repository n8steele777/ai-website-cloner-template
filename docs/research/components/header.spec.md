# Header Specification

## Overview
- Target area: fixed top navigation
- Interaction model: click-driven

## Structure
- Left-aligned Off Menu logo
- Right-aligned theme toggle and dot-grid menu toggle
- Expandable panel containing primary navigation and resource links

## Styles
- Padding: `16px` mobile, `24px` desktop
- Foreground-driven icon styling
- Floating background panel with blur and soft shadow

## Behaviors
- Theme toggle flips root color mode
- Menu toggle expands/collapses the panel
- `Escape` closes the panel

## Responsive
- Same overall structure at all breakpoints
- Panel stays anchored to the header control cluster
