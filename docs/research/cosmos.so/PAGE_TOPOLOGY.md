### Layout Notes

- The page is a single vertical flow with a persistent sticky header shell.
- Visual hierarchy depends on oversized type, broad gutters, and collage/image groupings.
- The reference page uses white space rather than card-heavy segmentation.

### Interaction Model

- Header: sticky, lightweight, mobile menu toggle
- Hero: static composition with CTA interactions
- Film CTA: hoverable/clickable row
- Editorial sections: mostly static with subtle hover states only
- Footer CTA: static
