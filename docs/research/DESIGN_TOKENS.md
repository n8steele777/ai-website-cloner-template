# Off Menu Homepage Design Tokens

## Colors
- Light background: `#fdfdfc`
- Light foreground: `#12110d`
- Dark background: `#12110d`
- Dark foreground: `#edecec`
- Border light: `rgba(18, 17, 13, 0.10)`

- Muted text dark: `rgba(237, 236, 236, 0.55)`

## Typography
- Primary family: `Neue Montreal`
- Safari fallback family: `Neue Montreal Safari`
- Hero kicker/headline: `14px`, `500`, `14px line-height`
- Title overlay: `36px` mobile, `40px+` desktop, `500`, tight tracking
- Menu links: `24px`, `500`
- Contact copy: `18px` mobile, `20px` desktop

## Spacing
- Global page padding: `16px` mobile, `24px` desktop
- Header controls: `40px square`
- Overlay controls: `40px` mobile, `48px` desktop
- Footer section: `24px` mobile, `32px` desktop

## Radius & Surfaces
- Orbit bubbles: fully rounded
- Menu panel radius: `12px`
- Buttons: pill or circle depending on role

## Motion
- Theme transitions: `300ms ease`
- Menu open/close: `300ms ease-in-out`
- Hero word reveal: staggered upward entrance
- Active title transition: `~450ms`, `power3.out`
- Orbit bubble idle motion: subtle sine yoyo drift
